# Telegram бот для розыгрышей

## Описание
Telegram бот для проведения розыгрышей с возможностью создания конкурсов, управления участниками и определения победителей.

## Основные возможности
- Создание и управление розыгрышами
- Проверка подписки на каналы
- Система рефералов
- Административная панель
- Статистика участников и розыгрышей
- Рассылка сообщений

## Технологии
- aiogram 3.x
- SQLAlchemy 2.x
- PostgreSQL
- Redis
- APScheduler
